## Intro
Simple project using nestjs+prisma

## Installation and configuration

```bash
$ yarn install
$ yarn prisma migrate dev --name init
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```
